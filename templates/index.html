Movies Recommendation System
Using Graph Neural Network

The Movie Recommender System is an important problem because these tasks are widely used for movie recommendations by services like Netflix or Amazon Prime video. There have been numerous efforts to build a recommender system using Collaborative Filtering, Neural Networks based on Embeddings, Neural Collaborative Filtering, etc. We introduce a Graph Neural Network (GNN) based recommender system.
